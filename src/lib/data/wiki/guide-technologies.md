# Guide des Technologies : L'Arbre de la Connaissance

> _"Celui qui cesse de rechercher cesse de progresser. Et celui qui cesse de progresser est déjà mort."_
> — Axiome des Laboratoires Omni

---

## Introduction : La Recherche & Développement

Dans le Secteur Omni, la technologie est le **multiplicateur de puissance**. Une machine de niveau supérieur produit plus vite, consomme moins, et te donne un avantage sur tes concurrents.

Mais les technologies avancées ne sont pas données. Elles doivent être **recherchées** — un investissement en temps et en argent qui définira la trajectoire de ton empire.

---

## L'Arbre Technologique

### Structure

L'arbre technologique est organisé en **Tiers** correspondant à des niveaux de complexité :

| Tier | Niveau Requis | Exemples                              |
| ---- | ------------- | ------------------------------------- |
| 1    | 1-2           | Automatisation de base, Métallurgie   |
| 2    | 3-5           | Extraction avancée, Énergie thermique |
| 3    | 5-7           | Acier, Verre                          |
| 4    | 8-10          | Pétrochimie, Plastique                |
| 5    | 15-20         | Électronique, Haute technologie       |

### Les Prérequis

Chaque technologie a des **prérequis** — des technologies qu'il faut débloquer avant :

```
Métallurgie de Base
    └── Forge d'Acier
            └── Ligne d'Assemblage
                    └── Électronique
                            └── Usine High-Tech
```

Tu ne peux pas sauter les étapes. Planifie ta progression !

## Les Technologies par Tier

### Tier 1 : Les Fondations

**Automatisation de Base** (Niveau 1)

- Coût : 5 000₭
- Débloque : Exploitation Forestière, Mine de Charbon, Scierie

C'est ta première recherche. Elle ouvre les portes de la production industrielle.

**Métallurgie de Base** (Niveau 2)

- Coût : 20 000₭
- Prérequis : Automatisation de Base
- Débloque : Fonderies (fer, cuivre), Extracteurs avancés

Le fer et le cuivre sont les briques de toute industrie. Cette technologie est essentielle.

### Tier 2-3 : L'Industrie

**Énergie Thermique** (Niveau 5)

- Coût : 50 000₭
- Prérequis : Métallurgie
- Débloque : Centrale Thermique

Ta première source d'énergie industrial-scale. Nécessaire pour les machines gourmandes.

**Production de Verre** (Niveau 5)

- Coût : 75 000₭
- Prérequis : Métallurgie
- Débloque : Four à Verre

Le verre est un composant clé de nombreux produits avancés.

**Forge d'Acier** (Niveau 7)

- Coût : 150 000₭
- Prérequis : Énergie Thermique
- Débloque : Presse à Acier

L'acier est le squelette de l'industrie lourde. Indispensable pour les constructions avancées.

### Tier 4 : La Pétrochimie

**Extraction Pétrolière** (Niveau 8)

- Coût : 200 000₭
- Prérequis : Acier
- Débloque : Pompe à Pétrole

Le pétrole brut est la base de nombreux produits modernes.

**Ère du Plastique** (Niveau 10)

- Coût : 500 000₭
- Prérequis : Extraction Pétrolière
- Débloque : Raffinerie, Plastique

Le plastique est partout : électronique, emballages, composants. Incontournable.

**Plateforme Offshore** (Niveau 12)

- Coût : 2 000 000₭
- Prérequis : Plastique
- Débloque : Plateforme Offshore

L'extraction massive. Nécessite une chaîne de production complète en amont.

### Tier 5 : La Haute Technologie

**Ligne d'Assemblage** (Niveau 15)

- Coût : 10 000 000₭
- Prérequis : Plastique
- Débloque : Moteurs Électriques, Assemblage complexe

La première étape vers les produits finis de haute valeur.

**Électronique** (Niveau 18)

- Coût : 12 000 000₭
- Prérequis : Assemblage
- Débloque : Circuits, Processeurs

Les composants du monde numérique.

**Usine High-Tech** (Niveau 20)

- Coût : 15 000 000₭
- Prérequis : Électronique
- Débloque : Smartphones, Ordinateurs

Le sommet de l'arbre. Les produits les plus rentables du Secteur Omni.

## Le Processus de Recherche

### Lancer une Recherche

1. Ouvre le panneau des Technologies
2. Sélectionne une technologie débloquée (prérequis remplis)
3. Vérifie que tu as le niveau et les fonds requis
4. Lance la recherche

### Le Temps de Recherche

Certaines technologies sont instantanées. D'autres prennent du temps :

```
Temps de recherche = Valeur définie dans la technologie
```

Pendant ce temps, les fonds sont bloqués et tu ne peux pas annuler.

### Les Items Requis

Certaines technologies avancées nécessitent des **items** en plus de l'argent :

```
Plateforme Offshore :
- 50 000 Acier
- 20 000 Plastique
- 5 000 Processeurs
```

Ces items sont consommés à l'achat de la technologie.

## Stratégies de Recherche

### La Ligne Directe

Concentre-toi sur une seule branche de l'arbre :

```
Base → Métallurgie → Acier → Pétrole → Plastique → Assemblage → High-Tech
```

Avantages : Progression rapide vers le sommet
Inconvénients : Branches latérales négligées

### L'Élargissement

Débloque toutes les technologies de chaque tier avant de monter :

```
Tier 1 : Tout
Tier 2 : Tout
Tier 3 : Tout
...
```

Avantages : Polyvalence maximale
Inconvénients : Progression plus lente vers le haut

### L'Hybride

Priorise les technologies critiques, élargis quand nécessaire.

## Les Items Débloqués

Chaque technologie débloque des **items** que tu peux ensuite :

- Acheter sur le marché (si disponible)
- Produire dans tes usines
- Utiliser dans d'autres recettes

### Exemple : Métallurgie de Base

Débloque :

- Fonderie Simple (machine)
- Fonderie Cuivre (machine)
- Lingot de Fer (item)
- Lingot de Cuivre (item)
- Extracteur Fer (machine)
- Extracteur Cuivre (machine)

## L'Économie de la Recherche

### Le Coût Réel

Le coût affiché n'est pas le coût réel. Calcule aussi :

- Le temps de recherche (temps = argent)
- Les items requis (valeur marchande)
- Le coût d'opportunité (tu aurais pu investir ailleurs)

### Le ROI de la Recherche

Après avoir débloqué une technologie, calcule combien de temps il faut pour rentabiliser :

```
Temps de rentabilisation = (Coût total) / (Profit supplémentaire par période)
```

Si le temps est trop long, la technologie n'était peut-être pas prioritaire.

## Les Catégories de Technologies

### Resource (Ressources)

Technologies liées à l'extraction et au traitement des matières premières.

### Industry (Industrie)

Technologies liées à la transformation et à la production.

### Tech (Technologie)

Technologies liées à l'énergie et à l'électronique.

## Conseils Avancés

### Anticipe tes Besoins

Regarde 2-3 technologies en avant. De quoi auras-tu besoin ? Commence à accumuler les items et les fonds.

### Ne Recherche Pas au Hasard

Chaque recherche doit avoir un objectif précis : débloquer une machine rentable, accéder à une ressource critique, ou préparer le tier suivant.

### Surveille le Marché

Parfois, il est plus rentable d'acheter un produit que de débloquer la technologie pour le produire. Calcule avant de rechercher.
