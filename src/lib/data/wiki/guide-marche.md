# Guide du Marché : L'Économie du Secteur Omni

> _"Le marché ne ment jamais. Il te dit exactement ce que tout le monde pense — et combien ils sont prêts à payer pour ça."_
> — Axiome du Marionnettiste

---

## Introduction : Le Sang de l'Empire

L'argent est le sang du Secteur Omni. Sans lui, tes usines s'arrêtent, tes employés partent, ton empire s'effondre. Et l'argent, tu le gagnes sur le **marché**.

Le marché est l'arène où les PDG s'affrontent — non pas avec des armes, mais avec des prix.

---

## Les Bases du Commerce

### Acheter

Le marché te permet d'acheter des items que tu ne produis pas (encore) :

1. Ouvre le Marché
2. Sélectionne l'item désiré
3. Vérifie le prix et ta trésorerie
4. Confirme l'achat

L'item apparaît instantanément dans ton inventaire.

### Vendre

Les items dans ton inventaire peuvent être vendus :

1. Ouvre ton Inventaire
2. Sélectionne l'item à vendre
3. Vérifie le prix du marché
4. Confirme la vente

L'argent est crédité immédiatement.

---

## Les Prix du Marché

### Le Système de Prix Fixes

Dans le Secteur Omni, chaque item a un **prix fixe** défini par le système. Ce prix ne fluctue pas — il est identique pour tous les PDG, à tout moment.

**Avantages du système fixe** :

- Prévisibilité totale des revenus
- Calcul de rentabilité simplifié
- Pas de spéculation (pas de stress)

**Implications stratégiques** :

- La différence se fait sur les **coûts de production**, pas sur le timing de vente
- Celui qui produit le moins cher gagne le plus
- L'efficacité opérationnelle est reine

### Le Spread

Il y a généralement une différence entre le prix d'achat et le prix de vente :

```
Prix Achat > Prix Vente
```

Le marché prend une marge. C'est normal.

---

## Disponibilité des Items

### Items Disponibles

Tous les items ne sont pas achetables sur le marché. Seuls ceux marqués `MarketAvailable: true` le sont.

**Disponibles** : Bois, Pierre, Fer, Cuivre, Charbon, machines de base...

**Non disponibles** : Plateforme Offshore, Acier Renforcé (doivent être craftés)

### Implications Stratégiques

Les items non disponibles sur le marché ont une valeur stratégique supérieure — tu es le seul à pouvoir les produire.

---

## L'Inventaire

### Structure

Ton inventaire est divisé par type d'item :

- **Ressources Brutes** : Bois, Pierre, Minerais...
- **Composants** : Lingots, Plastique, Verre...
- **Produits Finis** : Moteurs, Smartphones...
- **Machines** : Tes équipements en stock

### Capacité

L'inventaire de base n'a pas de limite stricte, mais les stockages dans l'usine ont des capacités :

- **Petit Entrepôt** : 1000 unités (solides)
- **Citerne Standard** : 1000 litres (liquides)

## Stratégies de Marché

### La Production Intégrée

Avec des prix fixes, la clé du profit est de **produire toi-même** plutôt que d'acheter :

```
Coût Production = Coût Ressources + Coût Main-d'œuvre + Coût Énergie
Si Coût Production < Prix Marché → Produis
Sinon → Achète
```

L'objectif est de minimiser ton coût de revient pour maximiser ta marge.

### Le Monopole de Niche

Concentre-toi sur un item que peu de corporations produisent. Deviens le fournisseur dominant.

### L'Intégration Verticale

Extrais → Transforme → Vends. Chaque étape que tu maîtrises augmente ta marge :

```
Marge Totale = Prix Vente Final - Coût Extraction Initial
```

Plus tu contrôles de la chaîne, plus tu gardes de valeur.

---

## Le Calcul de Rentabilité

### Marge Brute

```
Marge = Prix de Vente - Coût de Revient
```

### Marge sur Production

```
Coût Revient = Σ(Coût des entrées) + (Temps × Coût horaire machine)
```

### ROI du Craft

```
ROI = Marge / Coût Revient × 100%
```

Un ROI de 50% signifie que tu gagnes 50₭ pour chaque 100₭ investi.

## L'Économie de l'Énergie

### Coût Énergétique

L'énergie a un coût, même si elle est "gratuite" (manuelle) :

- **Manuelle** : Coût = Salaire des employés
- **Thermique** : Coût = Prix du charbon
- **Solaire/Éolienne** : Coût = Amortissement des installations

### Impact sur les Marges

```
Marge Nette = Marge Brute - Coût Énergie
```

Une machine qui consomme trop peut avoir une marge négative malgré un prix de vente élevé.

---

## La Bourse (Avancé)

### Les Actions

Dans le Secteur Omni, les corporations peuvent être cotées en bourse. La valeur de l'action reflète la perception de la santé de l'entreprise.

### Le Trading

Les PDG avancés peuvent :

- Acheter des actions d'autres corporations (investissement)
- Vendre des parts de leur propre corporation (lever des fonds)
- Shorter une corporation (parier sur sa chute)

### Les Dividendes

Les corporations rentables versent des dividendes aux actionnaires. C'est un revenu passif.

---

## Les Pièges à Éviter

### L'Illusion de la Marge

Un item se vend 100₭, tu l'as produit avec des ressources à 50₭. Marge = 50₭ ? Non !

N'oublie pas :

- Coût des employés
- Coût de l'énergie
- Usure des machines
- Temps d'immobilisation du capital

### La Dépendance au Marché

Acheter tout sur le marché réduit tes marges. Intègre verticalement quand possible.

## Conclusion

Le marché du Secteur Omni est simple : les prix sont fixes. La compétition se fait sur l'**efficacité**. Celui qui produit le moins cher gagne le plus.

Optimise tes coûts. Intègre ta chaîne. Domine par l'efficacité.

> _"Le PDG qui ignore le marché sera ignoré par le marché."_
> — Vérité du Secteur Omni
